# Horizontal  scrolling gallery - ScrollTrigger

A Pen created on CodePen.

Original URL: [https://codepen.io/GreenSock/pen/dydpJzY](https://codepen.io/GreenSock/pen/dydpJzY).

Demonstrates the "containerAnimation" feature in ScrollTrigger (new in 3.8.0) which allows you to implement ScrollTriggers on elements that are being fake-scrolled into the viewport via a GSAP animation.
